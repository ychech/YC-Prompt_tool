使用说明

快速开始

  1. 选择数据文件：首次使用点击"选择文件"创建或打开 JSON 数据文件
  2. 创建提示词：点击"新建提示词"按钮创建你的第一个提示词
  3. 保存数据：点击"保存"按钮或开启"自动保存"功能

数据存储

  • 文件模式：数据直接保存到 JSON 文件，无容量限制
  • 临时模式：数据仅在当前会话有效，刷新页面会丢失
  • 自动保存：开启后每 30 秒自动保存（需先选择文件）
  • 导入/导出：支持 JSON 格式的数据导入和导出

分类管理

  • 内置分类：写作、编程、分析、创意、翻译、其他
  • 新建分类：点击分类列表底部的"新建分类"按钮
  • 删除分类：鼠标悬停分类项，点击右侧的 × 删除
  • 删除分类后，该分类下的提示词会自动移到"其他"分类

标签管理

  • 添加标签：编辑提示词时，在标签输入框输入或选择已有标签
  • 新建标签：点击标签区域标题右侧的 + 按钮
  • 删除标签：点击自定义标签上的 × 删除
  • 标签筛选：点击标签可筛选包含该标签的提示词

精选提示词

  • 点击提示词卡片上的星星图标设为精选
  • 精选提示词会显示在"精选"分类中
  • 再次点击星星可取消精选

版本历史

  • 每次编辑提示词会自动保存历史版本
  • 点击提示词卡片上的"历史"按钮查看版本列表
  • 最多保留 10 个历史版本
  • 支持预览和恢复到任意历史版本

Markdown 编辑器

  • 支持 Markdown 语法格式化提示词内容
  • 工具栏提供快捷按钮：粗体、斜体、标题、列表、代码块等
  • 点击眼睛图标可切换预览/编辑模式
  • 使用 {{变量名}} 格式定义变量

快捷键

  • Ctrl/Cmd + S：保存数据到文件
  • Ctrl/Cmd + N：新建提示词
  • Esc：关闭弹窗/退出全屏编辑
  • Enter：在标签输入框中按回车添加新标签
  • Backspace：在标签输入框中按退格删除最后一个标签

使用技巧

  • 在侧边栏选择分类后，新建提示词会自动使用该分类
  • 点击提示词卡片中的标签也可进行筛选
  • 使用搜索框可同时搜索标题、内容和标签
  • 网格/列表视图切换可改变提示词展示方式
