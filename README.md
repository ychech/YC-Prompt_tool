# YC 提示词工具

一个简洁、高效的提示词（Prompt）管理工具，支持分类、标签、版本历史、Markdown 编辑器等功能。

🌐 **在线访问**: https://ychech.github.io/YC-Prompt_tool/

---

## ✨ 轻版特点与优势

### 核心优势

| 特性 | 说明 |
|------|------|
| **零部署** | 单个 HTML 文件，双击即可使用 |
| **零成本** | GitHub Pages 免费托管，无需服务器 |
| **隐私优先** | 数据存储在本地，不上传云端 |
| **离线可用** | 打开后断网也能正常使用 |
| **跨平台** | 任何有浏览器的设备都能运行 |

### 功能亮点

- 📁 **分类管理** - 内置分类 + 自定义分类
- 🏷️ **标签系统** - 灵活标签，支持筛选
- 📝 **Markdown 编辑器** - 实时预览，工具栏快捷操作
- 🌟 **精选功能** - 收藏常用提示词
- 🕐 **版本历史** - 自动保存历史版本，支持恢复
- 🔍 **全文搜索** - 支持标题、内容、标签搜索
- 💾 **多种存储方式** - 文件存储 / 浏览器本地存储

### 浏览器支持

| 浏览器 | 实时保存 | 说明 |
|--------|---------|------|
| Chrome/Edge | ✅ 支持 | 推荐使用，可直接保存到原文件 |
| Safari/Firefox | ⚠️ 下载模式 | 保存时下载新文件，数据自动备份到本地 |

## 🗺️ 数据库版本规划

### 为什么要做数据库版？

当前轻版适合个人使用，但存在以下局限：

| 轻版局限 | 数据库版解决方案 |
|---------|----------------|
| 换设备需手动迁移数据 | 账号登录，云端自动同步 |
| 多设备使用麻烦 | 任何设备登录即同步最新数据 |
| 大量数据时浏览器性能下降 | 数据库存储，服务端查询 |
| 团队协作困难 | 共享工作空间，权限管理 |
| 不支持复杂搜索 | 全文检索、智能推荐 |

### 技术方案

采用 **Cloudflare Workers + D1 (SQLite)** 架构：

- **全球 CDN** - 访问速度快
- **免费额度充足** - 10万次请求/天
- **零服务器维护** - 无需运维
- **OAuth 登录** - 支持 GitHub/Google 账号

### 两个版本并存

| 版本 | 定位 | 适用场景 |
|------|------|---------|
| **轻版** | 免费、快速、个人 | 个人用户、隐私敏感、离线使用 |
| **数据库版** | 专业、协作、团队 | 团队协作、多设备同步、大量数据 |

**数据互通** - 轻版导出的 JSON 可导入数据库版，反之亦然。
