# YC 提示词工具

一个简洁、高效的提示词（Prompt）管理工具。

🌐 **在线访问**: https://ychech.github.io/YC-Prompt_tool/

---

## 三个版本

### 版本1：轻版（当前）- Chrome/Edge Only

**特点：**
- ✅ 仅支持 Chrome/Edge 浏览器
- ✅ 直接读写本地文件（不是下载）
- ✅ 零部署，单个 HTML 文件
- ✅ 零成本，GitHub Pages 免费托管
- ✅ 数据完全本地，隐私安全

**使用方式：**
1. 用 Chrome 或 Edge 打开
2. 创建新文件或打开已有 JSON 文件
3. 编辑后按 Ctrl+S 直接保存到原文件

**非 Chrome/Edge 用户：**
> 页面会显示阻止提示，引导下载 Chrome 或 Edge

---

### 版本2：数据库版（规划中）

**目标：** 支持所有浏览器，云端同步

**技术栈：**
- 前端：React/Vue + Tailwind CSS
- 后端：Cloudflare Workers
- 数据库：Cloudflare D1 (SQLite)
- 认证：GitHub/Google OAuth

**功能：**
- [ ] 账号登录，数据云端同步
- [ ] 任何浏览器都能使用
- [ ] 多设备自动同步
- [ ] 团队协作空间
- [ ] 权限管理
- [ ] 分享链接

**部署成本：** 免费（Cloudflare 免费额度）

---

**目标：** 原生应用体验，完全离线

## 功能特性

- 📁 **分类管理** - 内置分类 + 自定义分类
- 🏷️ **标签系统** - 灵活标签，支持筛选
- 📝 **Markdown 编辑器** - 实时预览，工具栏快捷操作
- 🌟 **精选功能** - 收藏常用提示词
- 🕐 **版本历史** - 自动保存历史版本
- 🔍 **全文搜索** - 支持标题、内容、标签搜索
- 💾 **实时保存** - Ctrl+S 直接保存到文件
