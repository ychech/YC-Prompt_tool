# YC 提示词工具

一个简洁、高效的提示词（Prompt）管理工具，支持三种使用方式。

🌐 **在线访问**: https://ychech.github.io/YC-Prompt_tool/

---

## 三个版本

### 版本1：轻版（当前）⭐

**特点：**
- ✅ **仅 Chrome/Edge** - 其他浏览器会显示阻止提示
- ✅ **直接保存** - Ctrl+S 直接写入原文件（不是下载）
- ✅ **零成本部署** - GitHub Pages 免费托管
- ✅ **数据本地** - 隐私安全，不上传云端

**适用：** 个人使用，有 Chrome/Edge 的用户

**在线使用：** https://ychech.github.io/YC-Prompt_tool/

---

### 版本2：数据库版（开发中）

**特点：**
- ✅ **所有浏览器** - Safari/Firefox/Chrome/Edge 全支持
- ✅ **云端同步** - 登录账号，数据自动同步
- ✅ **多设备** - 手机/平板/电脑数据互通
- ✅ **团队协作** - 共享工作空间

**技术栈：** Cloudflare Workers + D1 + Vue 3

**状态：** 📝 API 和数据库设计完成，前端开发中

**目录：** [`v2-database/`](./v2-database/)

---

### 版本3：桌面版（规划中）

**特点：**
- ✅ **原生应用** - Windows/Mac/Linux 三平台
- ✅ **完全离线** - 无需网络，本地文件
- ✅ **系统集成** - 双击 .json 文件打开
- ✅ **快捷键** - 系统级快捷键支持

**技术栈：** Tauri (Rust) + Vue

**状态：** 📝 技术调研完成，待开发

**目录：** [`v3-desktop/`](./v3-desktop/)

---

## 快速选择

| 你的需求 | 推荐版本 | 原因 |
|---------|---------|------|
| 只用 Chrome/Edge，要简单 | **轻版** | 零部署，直接保存到文件 |
| 用 Safari/Firefox，要同步 | **数据库版** | 等 v2 完成，或先用其他工具 |
| 要完全离线，原生体验 | **桌面版** | 等 v3 完成 |
| 团队协作 | **数据库版** | 支持共享和权限管理 |
